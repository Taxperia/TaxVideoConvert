## Responsible Disclosure

We kindly ask that you:

- Do not publicly disclose the vulnerability before it has been resolved.
- Allow reasonable time for investigation and patching.
- Avoid exploiting the issue beyond proof-of-concept testing.

We appreciate responsible disclosure and security contributions from the community.
